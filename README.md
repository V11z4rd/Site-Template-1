## 🎯 Objetivo

Template de site para divulgação profissional e associação a contas em distintas plataformas. Ideal para profissionais autônomos, pequenos negócios e empreendedores que desejam uma presença digital simples e eficaz.

---

## ✨ Características

- ✅ Layout simples e limpo
- ✅ Totalmente responsivo
- ✅ Estrutura HTML semântica
- ✅ Fácil customização
- ✅ Integração com redes sociais
- ✅ Otimizado para SEO básico
- ✅ Sem dependências externas (HTML/CSS puro)

## 📁 Estrutura do Projeto

```
templateSite1/
├── index.html              # Página principal
├── README.md               # Este arquivo
├── css/
│   ├── reset.css           # Reset de estilos CSS padrão
│   ├── global.css          # Estilos globais do site

│       └── footer.css      # Estilos do rodapé
```

## 🚀 Como Usar

1. **Clone ou faça o download** deste repositório
2. **Customize o conteúdo** em `index.html`:
   - Altere o nome e profissão
   - Adicione sua foto na pasta `image/photo/`
   - Atualize os links das redes sociais
   - Modifique as informações de localização

3. **Personalize os estilos** em `css/`:
   - Altere cores em `global.css`
   - Ajuste layouts em `components/header.css` e `components/footer.css`

4. **Abra o arquivo** `index.html` no seu navegador ou hospede em um servidor web
